## 🔧 두 가지 설계 접근법

### 1. 공학 기술 기반의 설계

**핵심 개념**: 정사면체 구조 + 외부 연장 봉 시스템

**주요 특징**:
- **정사면체 중심부** (80mm 변 6개) - 계란 고정 및 충격 분산
- **외부 연장 봉** (280mm 봉 6개) - 각 변을 3.5배 연장하여 충격 흡수
- **내부 지지대** (12개) - 구조 강성 유지
- **테이프 면** (24개) - 에너지 소산

**재료**:
- 정사면체: 나무젓가락
- 외부 봉: 플라스틱 봉
- 예상 충격 흡수율: 85-95%

### 2. 자유 설계

**핵심 개념**: 고무줄 서스펜션 + 다단계 보호 시스템

**주요 특징**:
- **400mm 정육면체 프레임** - 기본 골격
- **낙하산 시스템** - 낙하 속도 감소 (30-40%)
- **12개 고무줄 서스펜션** - 탄성 충격 흡수 (40-50%)
- **3단계 보호층** - 지점토 + 에어캡 + 솜

**재료**:
- 프레임: 플라스틱/나무 막대
- 서스펜션: 고무줄 12개
- 보호층: 지점토, 에어캡, 솜, 종이컵
- 예상 충격 흡수율: 70-80%

## 🛠️ 설치 및 사용

### 🚀 빠른 시작

#### 1. 저장소 클론
```bash
git clone https://github.com/zeetee1235/eggdrop.git
cd eggdrop
```

#### 2. 자동 빌드 (추천)
```bash
# Linux/macOS
cd scripts && chmod +x *.sh && ./build-all.sh

# Windows
cd scripts && build-pdf.bat
```

### 🪟 Windows 사용자

#### 필수 설치
- **Git**: https://git-scm.com/download/win
- **LaTeX (MiKTeX)**: https://miktex.org/download
- **FreeCAD**: https://www.freecad.org/downloads.php

#### 사용법
```cmd
cd scripts

# 환경 설정 확인
setup-windows.bat

# PDF 문서 생성 (모든 설계도)
build-pdf.bat

# FreeCAD 3D 모델 열기
open-freecad.bat

# 과제 요구사항 검증
run-tests.bat
```

#### 문제 해결
- **LaTeX 오류**: MiKTeX Console에서 "Install missing packages on-the-fly" 활성화
- **한글 경로 문제**: 프로젝트를 영문 경로로 이동 (예: `C:\Projects\eggdrop\`)
- **FreeCAD 경로**: 일반적으로 `C:\Program Files\FreeCAD X.X\bin\FreeCAD.exe`

### 🐧 Linux/macOS 사용자

#### 요구사항
- **FreeCAD**: 3D 모델 확인/생성
- **LaTeX**: PDF 문서 생성 (pdflatex, texlive)

#### 사용법
```bash
cd scripts

# 전체 프로젝트 빌드 (추천)
./build-all.sh

# 개별 실행
./run-freecad.sh                      # FreeCAD 3D 모델 생성/열기
./build-pdf.sh                        # LaTeX PDF 변환
./run-freecad.sh 공학_기술_기반의_설계  # 특정 설계만
./build-pdf.sh 자유_설계               # 특정 설계도만

# 도움말
./build-all.sh --help
```

#### 환경 설정 (선택사항)
```bash
# Ubuntu/Debian
sudo apt-get install texlive-latex-base texlive-fonts-recommended
sudo apt-get install freecad

# Fedora/CentOS
sudo dnf install texlive-latex freecad

# Flatpak (권장)
flatpak install flathub org.freecad.FreeCAD
```
